# Space-Survivors
This is a game about avoiding asteriods - simple ;)

There are 4 difficulties easy normal hard and ULTRA

You need to get the longest time on any difficulty you want to compete with yourself or your friends.

There is also an ability button designed to help you escape impossible situations, especially on ultra and hard modes.

The ability button helps you phase through rocks without taking damage and it has a base 5 sec cooldown BUT the amount you hold it that amount will be added to the cooldown to encourage quick bursts rather than long holds which make the game easier. There is also a warning before the timer runs out so that you can reposition. The base cooldown starts on spawn

Whenever you die there is a small time in which you are invincible you this to move to a spot where there is more area.

This is the PC version of my game which supports controller. If you want the mobile version then go to my itch page
